clj-ravendb.rest

rest-client

(rest-client url database {:keys [replicated? master-only-writes? enable-oauth? oauth-url oauth-expiry-seconds api-key ssl-insecure?], :or {replicated? false, master-only-writes? true, enable-oauth? true, oauth-expiry-seconds 600, ssl-insecure? false}})

Gets a client for a RavenDB endpoint at the given url and database.

Optionally takes a map of options. :replicated? is used to find replicated endpoints. :master-only-writes? is used to indicate that write operations only go to the master
